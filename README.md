# Recursive checkbox tree
A react recursive checkbox tree example for displaying and managing data in the folowing format:
```javascript
{
   tree: {
      one: {
         one_one: true,
         one_two: false
      },
      two: true,
      three: {
         three_one: {
            three_one_one: true,
            three_one_two: false
         }
      }
   }
}
```
## Preview
You can see it working [here](https://elpopisencio.github.io/checkbox-tree).
